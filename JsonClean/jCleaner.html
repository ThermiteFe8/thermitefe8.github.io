Welcome to the JSON Cleaner!

You may upload a json file you'd like to remove the values of OR paste it into the textfield below!

I don't recommend doing both, but if you do it'll prioritize the textfield.

Sometimes, things break if there straight up isn't any value associated with the keys/ids, so you can check the checkbox to add some placeholder text to your stripped JSON file

When you're ready, press "Clean JSON file!" to download!
